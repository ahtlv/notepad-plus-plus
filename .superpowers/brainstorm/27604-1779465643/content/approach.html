Чем будем строить?

Три варианта — с разными трейдоффами

🍎
Swift + AppKit
swiftc → .app bundle

Swift + AppKit Рекомендую

Пишем на Swift, компилируем через CLI без Xcode IDE. NSTextView — нативный macOS-редактор с автозаменой, спеллчекером и всем что надо из коробки. Итог: бинарник ~1-2MB, настоящая Mac-апка.

Плюсы

  • Всё нативное
  • Tiny size
  • Autocorrect из коробки
  • Быстрый старт

Минусы

  • Нужно немного Swift
  • Без Xcode — .app bundlе вручную
🦀
Tauri 2
Rust + HTML/CSS/JS

Tauri 2

Rust бэкенд + WebKit2 рендерит HTML-интерфейс. Популярный стек для десктопных апок. Надо ставить Rust + Node. Размер ~8-15MB. UI пишется как веб-страница.

Плюсы

  • Веб-стек (знакомо)
  • Красивый UI легко
  • Активное сообщество

Минусы

  • Надо ставить Rust
  • Overkill для блокнота
  • Autocorrect — руками
⚡
Electron
Node + Chromium

Electron

Node.js + встроенный Chromium. Самый привычный стек для десктопных веб-апок. Но это VS Code / Slack архитектура — финальный размер ~150MB+ для простого блокнота. Жирно.

Плюсы

  • Самый знакомый стек
  • Богатая экосистема

Минусы

  • ~150MB для блокнота??
  • Медленный старт
  • Жрёт память
